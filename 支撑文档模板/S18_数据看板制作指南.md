# 数据看板制作指南

## 什么是数据看板
数据看板（Dashboard）是将产品关键指标可视化展示的工具，帮助团队实时监控产品健康状态。

## 制作工具

|------|--------|------|------|------|

## 看板设计模板

### 管理层日报看板
| 区域 | 展示指标 | 图表类型 | 刷新频率 |
|------|---------|---------|---------|
| 顶部数字卡片 | DAU / 新增 / 收入 / 留存率 | 数字+环比箭头 | 日 |
| 左上 | DAU趋势（近30天） | 折线图 | 日 |
| 右上 | 渠道新增分布 | 饼图/柱状图 | 日 |
| 左下 | 留存曲线（1/7/30日） | 折线图 | 日 |
| 右下 | 收入趋势 | 柱状图+目标线 | 日 |

### 产品运营看板
| 区域 | 展示指标 | 图表类型 |
|------|---------|---------|
| 核心漏斗 | 访问→注册→激活→付费 | 漏斗图 |
| 功能热力图 | 各功能使用频次 | 热力图/柱状图 |
| 用户分群 | 活跃/沉默/流失占比 | 饼图 |

## 用AI生成看板设计方案

### Prompt模板
```
# Role
你是一位资深数据分析师，精通数据可视化设计。

# Task
请为 [产品名称] 设计一套数据看板方案，用于 [管理层日报/产品运营/增长分析]。

# Context
- 产品类型：[类型]
- 核心指标：[北极星指标]
- 数据工具：[神策/GrowingIO/自建]
- 查看人：[角色]

# Output
1. 看板布局图（文字描述各区域位置）
2. 每个区域的指标、图表类型、数据来源
3. 刷新频率和告警规则
4. 筛选条件（日期/渠道/用户群等）
```

## 快速上手：用飞书多维表格做看板
1. 打开飞书 → 多维表格 → 新建
2. 创建数据表，输入指标数据
3. 切换到「仪表盘」视图
4. 添加图表组件（折线/柱状/数字卡片）
5. 配置数据源和筛选条件
6. 分享给团队
