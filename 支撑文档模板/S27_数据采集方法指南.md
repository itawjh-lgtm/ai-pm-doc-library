# 数据采集方法指南

## 数据采集方式概览

| 采集方式 | 适用场景 | 工具 | 成本 |
|---------|---------|------|------|
| 全埋点（无代码） | 快速启动、基础行为 | GrowingIO | 中 |
| 代码埋点 | 精确的业务事件 | 神策/Mixpanel | 高 |
| 服务端埋点 | 核心业务数据 | 自建 | 高 |
| 爬虫采集 | 竞品数据、公开数据 | 八爪鱼/Scrapy | 低-中 |
| 问卷调研 | 用户态度/意愿 | 问卷星 | 低 |
| 第三方平台 | 行业报告数据 | 艾瑞/Statista | 低-高 |

## 一、产品行为数据采集

### 工具选择
| 数据平台 | 官网 | 适合阶段 | 核心功能 |
|---------|------|---------|---------|
| 神策分析 | https://www.sensorsdata.cn | 成长期+ | 埋点分析、用户画像 |
| GrowingIO | https://www.growingio.com | MVP+ | 无埋点+代码埋点 |
| Mixpanel | https://mixpanel.com | MVP+ | 事件分析、漏斗 |
| Amplitude | https://amplitude.com | 成长期+ | 行为分析、实验 |
| Google Analytics | https://analytics.google.com | Web | 免费全面 |
| Firebase | https://firebase.google.com | App | 免费基础分析 |

### 埋点实施流程
1. 产品定义指标 → 2. 编写埋点文档 → 3. 开发实施 → 4. 数据验证 → 5. 看板配置

## 三、用AI辅助数据分析

### Prompt模板
```
# Role
你是一位资深数据分析师。

# Task
请根据以下数据，帮我分析 [分析目标]。

# Data
[粘贴数据表格或描述数据特征]

# Output
1. 核心发现（3-5条，每条有数据支撑）
2. 异常点分析
3. 行动建议
```
