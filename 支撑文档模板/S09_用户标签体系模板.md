# 用户标签体系

## 标签体系概览
| 标签层级 | 说明 | 示例 |
|---------|------|------|
| 一级标签（大类） | 用户基本属性 | 人口属性、行为特征 |
| 二级标签（中类） | 具体维度 | 年龄段、活跃度 |
| 三级标签（标签值） | 具体取值 | 18-25岁、高活跃 |

## 完整标签清单
| 一级 | 二级 | 三级标签值 | 数据来源 | 更新频率 | 应用场景 |
|------|------|-----------|---------|---------|---------|
| 人口属性 | 年龄段 | 18以下/18-25/26-35/36-45/46+ | 注册信息 | 静态 | 画像分析 |
| | 性别 | 男/女/未知 | | | |
| | 城市级别 | 一线/新一线/二线/三线及以下 | | | |
| 行为特征 | 活跃度 | 高活(日活)/中活(周活)/低活(月活)/沉默 | 行为数据 | 每日 | 运营策略 |
| | 使用时段 | 早间/午间/晚间/深夜 | | | |
| 消费特征 | 付费状态 | 免费/试用/付费/过期 | 交易数据 | 实时 | 变现 |
| | 消费能力 | 高/中/低 | | | |

## 标签应用场景
| 场景 | 使用标签组合 | 目的 |
|------|------------|------|
| 精准推送 | 活跃度+使用时段 | 在合适时间触达 |
| 付费转化 | 活跃度+付费状态+消费能力 | 定向转化 |
